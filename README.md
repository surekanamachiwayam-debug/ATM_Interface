# ATM Interface

This project is a Java-based console application developed as part of the Oasis Infobyte Java Development Internship.

## Project Description
The ATM Interface project simulates basic banking operations such as deposit, withdrawal, money transfer, and transaction history management. The application uses Java concepts to create a simple and interactive ATM system.

## Features
- User Login Authentication
- Deposit Money
- Withdraw Money
- Transfer Funds
- View Transaction History
- Balance Management
- Console-Based User Interface

## Technologies Used
- Java
- ArrayList
- OOP Concepts
- Scanner Class

## How to Run
1. Open the project in any Java IDE.
2. Compile the program.
3. Run the `ATMInterface.java` file.
4. Enter the login credentials to access ATM operations.
